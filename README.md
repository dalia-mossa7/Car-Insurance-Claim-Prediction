Car Insurance Claim Prediction

This project is a Data Mining course project that we developed as a team as part of the course requirements. In this project, we applied machine learning techniques to solve a real-world problem in the insurance domain.

We followed the full Data Analytics Lifecycle, starting from problem definition and data collection, to data cleaning and preprocessing, exploratory data analysis (EDA), model building, and evaluation. The dataset includes demographic, behavioral, and financial features such as age, income, driving experience, accident history, vehicle type, and credit score.

We conducted all analysis and modeling using SAS Studio, where we performed data preprocessing, statistical analysis, and implemented different machine learning models.

During the data preparation phase, we handled missing values, corrected data types, and treated outliers to ensure data quality. We then performed exploratory analysis to understand patterns and relationships between variables.

We implemented multiple classification models including Decision Tree, Naïve Bayes, and Logistic Regression, and evaluated them using a composite metric based on accuracy, simplicity, lift, and stability. The best-performing model was a Decision Tree (Gini-based).

We also applied K-Means clustering to segment customers into different risk groups.

Conclusion:
We found that factors such as driving behavior and credit score can be reasonably relied upon in predicting insurance claims. However, these factors alone are not sufficient, and combining multiple features leads to more accurate and reliable predictions.
